# Lesson 12: Working with Strings

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Master Java's most-used data type — learn the essential String methods for searching, manipulating, and formatting text.

## What you'll learn
- String creation and immutability
- Essential methods: `length()`, `charAt()`, `substring()`, `indexOf()`, `contains()`, `replace()`, `split()`, `trim()`, `toUpperCase()`, `toLowerCase()`
- String comparison: `equals()` vs `==`
- StringBuilder for efficient concatenation
- String formatting with `printf` and `String.format()`

## Run the code
```bash
javac WorkingWithStrings.java
java WorkingWithStrings
```
