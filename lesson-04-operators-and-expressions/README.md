# Lesson 4: Operators and Expressions

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Learn how to perform calculations, compare values, and build logical expressions using Java's operators.

## What you'll learn
- Arithmetic operators: `+`, `-`, `*`, `/`, `%`
- Increment / decrement: `++`, `--`
- Assignment operators: `+=`, `-=`, `*=`, `/=`, `%=`
- Comparison operators: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logical operators: `&&`, `||`, `!`
- Ternary operator: `condition ? a : b`
- Short-circuit evaluation

## Run the code
```bash
javac OperatorsAndExpressions.java
java OperatorsAndExpressions
```
