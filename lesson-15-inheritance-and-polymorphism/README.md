# Lesson 15: Inheritance and Polymorphism

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Learn how classes can inherit from other classes to reuse code, and how polymorphism lets you write flexible, powerful programs.

## What you'll learn
- `extends` – inheriting from a parent class
- Method overriding with `@Override`
- `super` keyword – calling parent constructor/methods
- Polymorphism – treating different objects through a common type
- `instanceof` check
- `final` class and method

## Run the code
```bash
javac InheritanceAndPolymorphism.java
java InheritanceAndPolymorphism
```
