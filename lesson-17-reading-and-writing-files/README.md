# Lesson 17: Reading and Writing Files

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Learn to persist data beyond program execution — read from and write to files using Java's modern file I/O tools.

## What you'll learn
- Writing text files with `FileWriter` / `BufferedWriter`
- Reading text files with `FileReader` / `BufferedReader`
- Modern `Files` API (Java 11+) — `writeString`, `readString`, `readAllLines`
- Try-with-resources for automatic closing
- Appending to existing files
- Checking if a file exists

## Run the code
```bash
javac ReadingAndWritingFiles.java
java ReadingAndWritingFiles
```
> This program creates a file called `output.txt` in the current directory.
