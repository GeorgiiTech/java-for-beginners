# Lesson 11: User Input with Scanner

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Make your programs interactive — learn to read keyboard input from the user using Java's Scanner class.

## What you'll learn
- Importing and creating a Scanner
- Reading strings, integers, doubles, and booleans
- Input validation with loops
- Reading multiple values
- Closing the Scanner

## Run the code
```bash
javac UserInput.java
java UserInput
```
> Note: This program reads from standard input. You will be prompted to type values.
