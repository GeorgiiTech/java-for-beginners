# Lesson 18: Final Project — Student Grade Calculator

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Put everything you've learned together — build a fully functional, interactive Student Grade Calculator that uses OOP, collections, file I/O, user input, and error handling.

## What this project covers
- OOP: `Student` class with encapsulation
- `ArrayList` to store multiple students
- `Scanner` for interactive user input
- Error handling with try/catch
- File I/O — save results to a file
- Grade calculation logic
- Formatted output

## Run the code
```bash
javac StudentGradeCalculator.java
java StudentGradeCalculator
```

## Features
- Add multiple students with their scores
- Automatically calculates average, grade, and pass/fail
- Saves the report to `grade_report.txt`
- Handles invalid input gracefully
