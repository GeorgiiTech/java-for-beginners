# Lesson 16: Interfaces and Abstract Classes

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Learn two advanced OOP tools — interfaces define contracts, while abstract classes provide a middle ground.

## What you'll learn
- `interface` – defining a contract
- `implements` – fulfilling a contract
- Multiple interface implementation
- `abstract class` – partial implementation
- `abstract` methods
- Default interface methods (Java 8+)
- When to use interface vs abstract class

## Run the code
```bash
javac InterfacesAndAbstractClasses.java
java InterfacesAndAbstractClasses
```
