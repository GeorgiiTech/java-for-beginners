# Lesson 13: ArrayList and the Collections Framework

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Go beyond fixed-size arrays — learn ArrayList and other Java collections that grow and shrink dynamically.

## What you'll learn
- `ArrayList` – adding, removing, iterating
- `LinkedList` – when to prefer it
- `HashMap` – key-value storage
- `HashSet` – unique elements
- `Collections` utility class (sort, shuffle, min, max)
- Generics basics

## Run the code
```bash
javac ArrayListAndCollections.java
java ArrayListAndCollections
```
