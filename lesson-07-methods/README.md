# Lesson 7: Methods: Writing Reusable Code

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Learn to write methods — reusable blocks of code that keep your programs organized, clean, and easy to maintain.

## What you'll learn
- Defining and calling methods
- Parameters and return values
- Method overloading
- void vs return types
- Recursion basics

## Run the code
```bash
javac Methods.java
java Methods
```
