# Lesson 14: The static Keyword and Constants

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Understand one of Java's most misunderstood keywords — `static` — and learn to use constants with `final`.

## What you'll learn
- Static fields (class variables vs instance variables)
- Static methods
- Static initializer blocks
- Constants: `static final`
- Utility classes pattern
- When to use (and when NOT to use) static

## Run the code
```bash
javac StaticKeywordAndConstants.java
java StaticKeywordAndConstants
```
