# Lesson 9: Introduction to OOP: Classes and Objects

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Discover Object-Oriented Programming — the most important concept in Java — and learn to model real-world things as classes and objects.

## What you'll learn
- What is a class and an object
- Fields (instance variables) and methods
- Constructors
- `this` keyword
- Creating multiple objects
- Encapsulation with getters and setters

## Run the code
```bash
javac IntroToOOP.java
java IntroToOOP
```
