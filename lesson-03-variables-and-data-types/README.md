# Lesson 3: Variables and Data Types

> 📚 [Back to Course](https://georgii.tech/courses/java-for-beginners-from-zero-to-your-first-program) | [Course Repo](../../README.md)

Learn how to store and work with data in Java using variables, and discover the different types of data Java understands.

## What you'll learn
- Primitive types: `byte`, `short`, `int`, `long`, `float`, `double`, `char`, `boolean`
- Reference type: `String`
- Type casting (widening and narrowing)
- Constants with `final`
- Variable scope

## Run the code
```bash
javac VariablesAndDataTypes.java
java VariablesAndDataTypes
```
